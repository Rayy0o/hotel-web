# Hotel Booking Website Prototype

A modern, responsive web application for hotel room bookings and service requests.

## Features

- Room booking system with date selection and room type options
- Additional services request system (Room Service, Spa, Transportation, Concierge)
- Responsive design that works on all devices
- Modern UI with smooth animations and transitions
- Form validation and user feedback

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome for icons

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. No additional setup required as this is a frontend-only prototype

## Project Structure

```
hotel-booking/
├── index.html          # Main HTML file
├── styles.css          # CSS styles
├── script.js           # JavaScript functionality
└── README.md           # Project documentation
```
